About Me

My interest in computers and coding started when I was a kid. I remember thinking to myself "How does this computer work?" "what makes it work?" "Is it hard to build a computer?". So I just decided to open the computer up and find out myself at that moment it sparked my interest in computers, I wanted to know everything about them and not just the hardware I wanted to know what kind of code went into them as well. While looking up computer tutorial videos on youtube I started wondering how they designed such a website, What tools did they use, How difficult is it.

Years later I looked into it and found a website called codecademy where I started learning the 3 basic languages of software developing HTML, CSS and JAVASCRIPT. After highschool I enrolled in a trade school called Job Corps to learn IT in order to learn more about the hardware of computers(I had to do a TON of reading!) I found out that it's actually not that complicated to build computers. After I completed that school I enrolled in another trade school called "tech and media" to learn basic C programming (It was fun and I also got to learn some of the basics of unity as well). After I got my diploma from that school I later enrolled in another trade school called "General Assembly" and took an immersive course in software developing and that's where I learned alot more about HTML, CSS and JAVASCRIPT, As well as some new languages like SCSS(my favorite language), REACT and PYTHON.
